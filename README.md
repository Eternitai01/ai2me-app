# ai2me-app
AI2me.com - Your AI-powered search and assistant platform. Minimalistic. Intelligent. Fast.

## Features

- **Clean, Minimalistic UI** - Distraction-free interface focused on what matters
- **AI-Powered Search** - Get intelligent, contextual answers instantly
- **Lightning Fast** - Optimized for speed and performance
- **Modern Stack** - Built with Next.js 14, TypeScript, and Tailwind CSS

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide Icons
